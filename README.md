# Connected Battery CF App

Kostenlose lokale PWA für Material → Batch → Elektrode → C-Rate/CF-Rechnung.

## Sofort testen
Öffne `index.html` lokal im Browser.

## Kostenlos als Link über GitHub Pages
1. GitHub öffnen und ein neues **public** Repository erstellen, z. B. `battery-cf-app`.
2. Diese Dateien (`index.html`, `manifest.webmanifest`, `sw.js`, `icon.svg`) in das Repo hochladen.
3. Repository → **Settings** → **Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main**, Folder: **/root**.
6. Warten, bis GitHub den Link zeigt.
7. Link auf Android öffnen → Browser-Menü → **Zum Startbildschirm hinzufügen** / **App installieren**.

## Daten
Die App speichert Daten lokal im Browser (`localStorage`).
Nutze **Export JSON** und **Import JSON** für Backup oder Transfer zwischen PC und Handy.
